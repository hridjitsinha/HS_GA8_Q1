SQL for Data Extraction


-- Analyze customer purchase patterns
SELECT 
    c.customer_id,
    c.customer_name,
    COUNT(o.order_id) as total_orders,
    SUM(o.total_amount) as lifetime_value,
    AVG(o.total_amount) as avg_order_value
FROM customers c
LEFT JOIN orders o ON c.customer_id = o.customer_id
WHERE o.order_date >= DATE_SUB(CURDATE(), INTERVAL 1 YEAR)
GROUP BY c.customer_id, c.customer_name
HAVING total_orders > 5
ORDER BY lifetime_value DESC
LIMIT 100;
SQL is essential for data extraction. This query identifies high-value customers from the past year. Discuss the importance of efficient queries when working with large datasets.

Financial Mathematics

Compound Interest Formula

$$A = P\left(1 + \frac{r}{n}\right)^{nt}$$

Where: A = Final amount, P = Principal, r = Rate, n = Compounding frequency, t = Time

Sharpe Ratio (Risk-Adjusted Return)

$$S = \frac{R_p - R_f}{\sigma_p}$$

Where: Rp = Portfolio return, Rf = Risk-free rate, Ïƒp = Standard deviation

These formulas are fundamental in financial analysis. The compound interest formula shows exponential growth. The Sharpe ratio helps compare investment performance adjusted for risk. Higher Sharpe ratio indicates better risk-adjusted returns.

Statistical Foundations

Normal Distribution PDF

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}$$

Linear Regression (OLS)

$$\hat{\beta} = (X^TX)^{-1}X^Ty$$

Mean Squared Error

$$MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

These are core statistical concepts. The normal distribution is ubiquitous in nature and statistics. Linear regression via OLS is one of the most widely used modeling techniques. MSE is a standard metric for regression model evaluation.
